PASOS PARA GUARDAR UN PROYECTO
 COMANDOS // COMENTARIOS SOBRE EL COMANDO
1) git init //INICIO EL PROYECTO
2) git status //PARA SABER EL STATUS DE LOS ARCHIVOS
3) git add app.js // PARA AGREGAR UN ARCHIVO AL STAGE, EN ESTE CASO EL ARCHIVO ES "app.js", PERO SI PONGO "git add ." SE PASAN TODOS LOS ARCHIVOS NUEVO
4) git status //PARA ASEGURARNOS QUE SE AGREGARON TODOS LOS ARCHIVOS
5) git commit -m "mensaje" //PARA GUARDAR LOS CAMBIOS QUE YA ESTÁN AGREGADOS, EN EL MENSAJE (QUE DEBE IR EN INGLES) SE PONE UNA DESCRIPCION SOBRE LO QUE SE HIZO EN ESTA PARTE DEL CÓDIGO
6) ENTRAR EN GITHUB Y CREAR UN NUEVO REPOSITORIO
7) git remote add origin "URL" // PARA AGREGAR EL REPOSITORIO LOCAL QUE TENEMOS AL GITHUB
8) git branch -M main // PARA RENOMBRAR LA RAMA DE MASTER A main
9) git push -u origin main // SUBE LOS ARCHIVOS A GITHUB
